## 概要
Home / SymbolDetail / Comparison の軽微なUX改善に加え、追加で報告された不具合2件を修正しました。

## 追加修正（今回）
- SymbolDetail
  - AI論点カードの再生成時に `force_regenerate` を送信するよう変更
  - 1回目は生成できるが2回目以降は再生成されない問題を解消
- Symbols API
  - `POST /api/symbols/:symbolId/ai-summary/generate` に `force_regenerate` を追加
  - `force_regenerate=true` のときは重複スキップを行わず、新しい summary を生成
- Home
  - positions の銘柄名表示で、同一 `symbol_id` の watchlist 表示名を優先
  - `6758` のみ英語表記になるケースを回避

## 既存修正（前回）
- Home
  - positions の `symbol_id` 行を SymbolDetail へのリンク化
  - daily summary の表現を「AIデイリーサマリー」へ改善
- SymbolDetail
  - AI論点カードが available の場合でも再生成ボタンを表示
- walkthrough
  - Home / SymbolDetail / Comparison の確認手順を補足
