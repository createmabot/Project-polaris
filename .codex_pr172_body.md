## 概要
Home / SymbolDetail / Comparison の軽微なUX改善に加え、追加で報告された不具合4件を修正しました。

## 追加修正（今回）
- SymbolDetail 再生成
  - AI論点カード再生成時に `force_regenerate=true` を送信
  - 1回目のみ成功し2回目以降が再生成されない問題を解消
- Comparison 再生成
  - `POST /api/comparisons/:comparisonId/generate` に `force_regenerate` を追加
  - UIからの「AI比較総評を生成」は常に `force_regenerate=true` で実行
  - 旧英語サマリーの重複再利用を避けて新規生成
- 日本語出力の一貫性改善
  - symbol/comparison の deterministic 文面を日本語化
  - local_llm / openai_api の symbol/comparison system prompt を日本語強制に変更
  - request payload に `output_language: "ja"` を追加
- Home 表示名
  - positions の銘柄名表示で、同一 `symbol_id` の watchlist 表示名を優先
  - `6758` のみ英語表記になるケースを回避
